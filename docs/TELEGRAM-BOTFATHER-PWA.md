# Telegram (BotFather) + PWA: пошаговый запуск

В конце у вас будет:

- **Продакшен-ссылка** на CRM по **HTTPS** (обязательно для Telegram и PWA).
- **Бот в Telegram** с кнопкой меню **«Открыть CRM»** (или своим текстом), ведущей на ваш сайт — он откроется во встроенном браузере Telegram.
- **Установка на главный экран** телефона без App Store (PWA): ярлык, режим почти как у приложения.

Код уже вызывает `Telegram.WebApp.ready()` и `expand()` (компонент `TelegramWebAppInit`), чтобы окно в Telegram занимало доступную высоту.

---

## Шаг 0. Один раз: зафиксируйте продакшен-URL

1. Определите **точный адрес**, где у вас крутится CRM, например:  
   `https://crm.ваш-домен.ru`  
   или домен Vercel: `https://ваш-проект.vercel.app`
2. Откройте этот URL **в обычном браузере** на компьютере: должна открываться **страница входа** или дашборд без ошибки сертификата (замочек HTTPS).
3. Запомните **путь, с которого удобно заходить менеджерам**, чаще всего это:
   - `https://…/` или  
   - `https://…/login`  
   Для кнопки в боте лучше дать **прямую ссылку на вход**, если у вас основной экран защищён.

**Важно:** в BotFather и в Telegram можно указывать только **https://** (не `http://` и не IP без SSL).

---

## Шаг 1. Деплой и переменные (если ещё не сделали)

1. Залейте код на **Vercel** (или другой хостинг) из репозитория `crm-app`.
2. В панели проекта добавьте **все переменные** из `.env.example` / вашей локальной `.env.local` (Supabase URL, ключи и т.д.).
3. Дождитесь успешного **Deploy**, снова откройте прод-URL в браузере и войдите тестовым пользователем.

Пока прод не открывается стабильно с телефона в Chrome/Safari — дальше смысла нет.

## Шаг 2. Иконки PWA (уже в репозитории)

В репозитории лежат файлы:

- `public/pwa-icon-180.png`
- `public/pwa-icon-192.png`
- `public/pwa-icon-512.png`

Если вы **меняли логотип** `public/asiamix-logo.svg`, пересоберите иконки:

```bash
npm run pwa:icons
```

Закоммитьте PNG и задеплойте снова.

## Шаг 3. BotFather — создать бота (если ещё нет)

1. В Telegram откройте **@BotFather**.
2. Отправьте `/start`.
3. Отправьте `/newbot`.
4. Введите **отображаемое имя** (например `Asia Mix CRM`).
5. Введите **username бота** (должен оканчиваться на `bot`, например `AsiaMixCrmBot`).
6. BotFather пришлёт **токен** вида `123456:ABC-DEF...` — **не публикуйте** его и не коммитьте в git. Он нужен только если вы позже напишете **серверную** логику бота; для простой кнопки-ссылки токен можно только хранить у себя.

---

## Шаг 4. BotFather — кнопка меню со ссылкой на сайт (основной сценарий)

Так пользователи открывают вашу CRM **из чата с ботом** одним нажатием.

1. В чате с **@BotFather** отправьте `/mybots`.
2. Нажмите на **вашего бота** в списке.
3. Откройте **Bot Settings** (Настройки бота).
4. Найдите пункт **Menu Button** (Кнопка меню).
5. Выберите **Configure menu button** / **Edit menu button**.
6. BotFather спросит **текст кнопки** — например: `Открыть CRM` или `Asia Mix`.
7. Затем пришлите **URL** — ваш продакшен **HTTPS**, например:  
   `https://crm.ваш-домен.ru/login`
8. Дождитесь подтверждения от BotFather.

Проверка:

1. Найдите в Telegram **своего бота** по username.
2. Откройте чат с ботом → внизу слева возле поля ввода должна появиться **кнопка меню** (иконка или подпись).
3. Нажмите её — должна открыться **ваша CRM** внутри Telegram.

Если кнопки не видно: полностью **закройте чат с ботом** и откройте снова, либо перезапустите Telegram.

---

## Шаг 5. (Опционально) Кнопка «Web App» в сообщениях бота

Кнопка меню из шага 4 не требует программирования.  
Отдельный тип **Web App** на **inline-клавиатуре** настраивается **только через код** вашего бота (отправка сообщения с `web_app` URL). Если позже понадобится — это делается на сервере с токеном бота; для старта достаточно **Menu Button**.

---

## Шаг 6. PWA — «установить» на телефон без магазина

После деплоя откройте **тот же прод-URL** в браузере телефона.

### Android (Chrome)

1. Откройте сайт в **Chrome**.
2. Меню **⋮** → **«Установить приложение»** / **«Добавить на главный экран»** (формулировка может отличаться).
3. Подтвердите — на рабочем столе появится иконка **Asia Mix**.

### iPhone (Safari)

1. Откройте сайт в **Safari** (не внутри Telegram для этой операции удобнее именно Safari).
2. Кнопка **Поделиться** → **«На экран Домой»**.
3. Задайте имя ярлыка → **Добавить**.

Дальше ярлык открывает сайт **без адресной строки** (режим `standalone`), как отдельное приложение.

## Частые проблемы

**В Telegram белый экран или «не открывается»**  
- Проверьте, что URL начинается с **https://**.  
- Убедитесь, что деплой не упал и нет **Basic Auth**, блокирующего внешний доступ.  
- Откройте тот же URL в **Safari/Chrome** вне Telegram — если там ошибка, сначала чините прод.

**После входа выкидывает на логин**  
- Проверьте настройки **cookies** для вашего домена (на Vercel обычно ок).  
- Убедитесь, что не смешиваете два разных домена (www и без www) — лучше один канонический.

**Иконка на рабочем столе не та / старая**  
- Пересоберите иконки: `npm run pwa:icons`, закоммитьте PNG, задеплойте.  
- На iOS удалите старый ярлык и добавьте снова.

**В Telegram интерфейс «низкий», не на весь экран**  
- Уже вызывается `Telegram.WebApp.expand()`. Обновите приложение Telegram и откройте ссылку снова.
